MealMate is a Progressive Web App (PWA) built to help students at the University of South Carolina easily track their weekly meal plan swipes.  
It works on desktop and mobile, can be installed on iPhones via ‚ÄúAdd to Home Screen,‚Äù and functions completely offline using Workbox caching.

---

## Features
- Track up to 7 retail swipes per week  
- Automatically resets every Thursday at midnight  
- Saves progress locally with `localStorage`  
- Works offline and installable as a PWA  
- Responsive design for desktop and mobile  
- Easily extendable with Firebase or email automation

---

## Tech Stack
- HTML5, CSS3, JavaScript  
- Progressive Web App (PWA)  
- Workbox (for service worker + caching)  
- LocalStorage (can be upgraded to Firebase Firestore in the future)  
- GitHub Pages
